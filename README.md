# HRM front-end website

## Abtract:
## Using:
  + ReactJs
  + Typescript
## Libraries:
  + MUI (Material UI)
## Installation:
###  Install HRM FE with npm:
  ```bash
    npm install
  ```
### Install libraries:
  + MUI:
  ```bash
  npm install @mui/material @emotion/react @emotion/styled
  ```
## Run:
  ```bash
  npm run dev
  ```

  Copy đường dẫn rồi mở trên trình duyệt.
